Base of [StanislavPetrovV/DOOM-Level-Viewer](https://github.com/StanislavPetrovV/DOOM-Level-Viewer)

HP-DOOM is written in Python using HP Prime libraries. Tested with WAD files for DOOM1 and DOOM2
